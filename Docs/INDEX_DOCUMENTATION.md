### Scripts de Configuration

| Script | Windows | Linux/Mac | Description |
|--------|---------|-----------|-------------|
| Configuration .env | `setup-env.ps1` | `setup-env.sh` | Créer automatiquement les fichiers .env |
| Vérification | `verifier-configuration.ps1` | `verifier-configuration.sh` | Vérifier les prérequis et la configuration |

### Scripts de Démarrage

| Script | Windows | Linux/Mac | Description |
|--------|---------|-----------|-------------|
| Backend | `start-backend.bat` | `start-backend.sh` | Démarrer le serveur Laravel |
| Frontend | `start-frontend.bat` | `start-frontend.sh` | Démarrer le serveur React/Vite |

## 💡 Conseils

### Pour bien démarrer
1. ✅ Lisez [LISEZ_MOI_D_ABORD.txt](LISEZ_MOI_D_ABORD.txt)
2. ✅ Exécutez `verifier-configuration` pour vérifier les prérequis
3. ✅ Exécutez `setup-env` pour créer les .env
4. ✅ Suivez [DEMARRAGE_RAPIDE.md](DEMARRAGE_RAPIDE.md)

### Pour développer efficacement
1. 📖 Consultez [README.md](README.md) pour comprendre l'architecture
2. 🔍 Explorez `Mon_Miam_Miam/routes/api.php` pour voir les routes
3. 🧪 Lancez les tests pour vérifier votre code
4. 📝 Consultez les Policies pour comprendre les autorisations

### Pour résoudre des problèmes
1. 🔍 Consultez les logs : `Mon_Miam_Miam/storage/logs/laravel.log`
2. 📖 Cherchez dans la section "Dépannage" de la documentation
3. ⚙️ Vérifiez les variables .env
4. 🔄 Videz les caches Laravel : `php artisan cache:clear`

## 📝 Notes

- Tous les fichiers `.txt` peuvent être ouverts avec n'importe quel éditeur de texte
- Les fichiers `.md` (Markdown) sont mieux lus avec un lecteur Markdown ou sur GitHub
- Les scripts `.ps1` sont pour PowerShell (Windows)
- Les scripts `.sh` sont pour Bash (Linux/Mac)
- Tous les scripts sont documentés et commentés
